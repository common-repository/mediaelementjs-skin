== Description ==

The audio player skin from Premium Pixels ported to MediaElement.js.

### Usage

After activation the skin option should automatically update to using the custom skin. If not, go to Settings &gt; MediaElement.js and set the skin to `Custom`.

If you want to keep a different default skin, you can override individual players by setting the `skin` parameter like this:

	[video src="http://example.com/video.m4a" skin="mejskin"]

Check out <a href="http://wordpress.org/extend/plugins/media-element-html5-video-and-audio-player/">the MediaElement.js plugin</a> for more documentation on how to use the player plugin.

== Installation ==

1. First install and activate <a href="http://wordpress.org/extend/plugins/media-element-html5-video-and-audio-player/">MediaElement.js - HTML5 Audio and Video plugin</a>.
2. Install and activate this plugin.
3. Go to Settings &gt; MediaElement.js to configure the plugin.
